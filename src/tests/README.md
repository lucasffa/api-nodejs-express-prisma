# Testes

### Responsabilidade:
Garantir a qualidade do código e o correto funcionamento das funcionalidades.

### Função:
Executar conjuntos de instruções sobre o código para verificar se ele se comporta conforme o esperado em diferentes cenários.

### Interação:
Os testes são normalmente executados em ambientes de desenvolvimento e CI/CD antes das atualizações serem movidas para produção. Eles interagem com o código real, simulando comportamentos e verificando os resultados.
